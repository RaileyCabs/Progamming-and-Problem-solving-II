You need to create at least 4 java files. one for the main program, and three for each level
You are welcome to create more java files (such as random number generator, score calculator and others). 
No grade will be awarded if you use less than 4 java files in completion of this homework.                 
                

write a program that teaches arithmetic to a young child. the program tests addition and subtraction. In level 1 it tests only addition of numbers less than 10 whose sum is less than 10. in level 2, it tests addition
of arbitrary one digit numbers. in level 3 it tests subtraction of one digit numbers with a nonnegative difference

generate random problems and get the players input. the player gets up to two tries per problem. advance from one level to the
next when the player has achieved a score of five points
